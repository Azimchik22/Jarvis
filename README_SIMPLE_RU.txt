JARVIS AI (APK) SIMPLE PACK — без папки .github (чтобы Android не прятал)

Внутри есть файл:
- WORKFLOW_ANDROID_YML.txt  — это содержимое workflow для GitHub Actions.

ВАША ЗАДАЧА:
Создать на GitHub файл по пути:
.github/workflows/android.yml
и вставить туда содержимое из WORKFLOW_ANDROID_YML.txt

Дальше GitHub сам соберёт APK.

Остальные файлы проекта уже готовы (main.py, service.py, buildozer.spec и т.д.).
